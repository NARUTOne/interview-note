# P10

> JS 算法

**⚡题目**:

❓ 数组最长子序列 (不去重)

```js
var nums1 = [1]
var nums2 = [1,1]
//[1]
```

## 优解 🔥

需要注意数组长度，求最长子序列

> 空间换时间 哈希表，时间复杂度O(m + n) m为nums1长度，n为nums2长度

```js
const intersect = (nums1, nums2) => {
  const map = {}
  const res = []
  for (let n of nums1) {
    if (map[n]) {
      map[n]++
    } else {
      map[n] = 1
    }
  }
  for (let n of nums2) {
    if (map[n] > 0) {
      res.push(n)
      map[n]--
    }
  }
  return res
}
```
