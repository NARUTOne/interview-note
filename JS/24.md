# P24

> JS == 隐式类型转换

**⚡题目**:

❓ 题目如下

```js
var a = null;
if(a == 1 && a == 2 && a == 3){
  conso.log(1);
}
```

a 在什么情况下会打印 1 ?

## 优解 🔥

> 因为==会进行隐式类型转换 所以我们重写toString方法就可以了

```js
var a = {
  i: 1,
  toString() {
    return a.i++;
  }
}

if( a == 1 && a == 2 && a == 3 ) {
  console.log(1);
}
```

> valueOf, 对象对比模式

```js
let a = {
  i: 1,
  valueOf () {
    return a.i++
  }
}

if(a == 1 && a == 2 && a == 3) {
  console.log('1');
}
```

> 数组模式

```js
let a = [1,2,3];
a.toString = a.shift;
if( a == 1 && a == 2 && a == 3 ) {
  console.log(1);
}
```

> ES6 symbol

```js
let a = {[Symbol.toPrimitive]: ((i) => () => ++i) (0)};
if(a == 1 && a == 2 && a == 3) {
  console.log('1');
}
```
